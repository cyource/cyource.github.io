  • About this blog
    I created this blog in June 2016, using Jekyll - "a blog-aware, static site generator" (that's the official description), along with Google's mdl library and gh-pages. In order to locally run it, to test changes before pushing them live, I used Git on Bash on Ubuntu on Windows - read this post to see how this blog came to be. Though featuring posts on a variety of different topics (or intended to, anyway), the primary focus is, was, and will always be technology. The source code for this blog can be found here.

  • About me
    My name is Manas Rawat (aka "cyource"), and I'm a UK-based indepedent computer programmer and Android Developer with a passion for maths, science and programming, that loves comic book media (especially CW's The Flash and DC's Legends of Tomorrow). When I'm not coding away, I can usually be found on freenode IRC channels, asking for, or giving help.

  • Computer Programming
    I primarily code in object-orientated languages including Java (for GUI Android Apps), C++ (for Native Android command-line/terminal apps), C, C#, and some Python. I use styling, templating and markup languages for both web development - CSS, Liquid and HTML - and for the look and feel of GUI Android Apps - XML. In addition to these languages, I also have some experience with JavaScript. The source code for this blog and many of my projects can be found on my GitHub. I am also pretty well-versed in using Shell, Bash and ADB, and - as shown by the look and feel of this blog, my GUI Android Apps, and icons/logos I created - in user interface design.
