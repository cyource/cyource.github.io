  • About this blog
    I created River Logs (simply titled "Manas Rawat's Blog" back then) in June 2016, using Jekyll - "a blog-aware, static site generator" (that's the official description), along with Google's mdl library and gh-pages. In order to locally run it, to test changes before pushing them live, I used Git on Bash on Ubuntu on Windows - read this post to see how this blog came to be. Though featuring posts on a variety of different topics (or intended to, anyway), the primary focus will always be technology, whether it's app development, modding or Custom ROM power-using. The source code for this blog can be found here.

  • About me
    My name is Manas Rawat (aka "munowse" and "cyource"), and I'm a UK-based indepedent computer programmer and app developer with a passion for maths, science and programming, that loves comic book media (especially The CW's The Flash and DC's Legends of Tomorrow). When I'm not coding away, I'm usually either on freenode IRCs #android, #android-dev and #cyanogenmod channels, watching videos, talking to friends and playing games on my phone, reading (comics and books) or swimming.

  • Computer Programming
    I primarily code in object-orientated languages including Java, C#, C, C++ and some Python. For web development (i.e. that of this blog), I use (as most people do) HTML & CSS, with some Liquid. In addition to these languages, I also have a bit of experience with JavaScript. The source code for this blog and many of my projects can be found on my GitHub. As shown by the look and fell of my blog and apps, I'm quite well-versed in UI/X design.
